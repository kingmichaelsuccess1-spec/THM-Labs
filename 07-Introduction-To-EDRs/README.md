## Description
This room introduces Endpoint Detection and Response (EDR) — one of the most critical tools in a modern SOC analyst's toolkit. It covers how EDRs work, how they differ from traditional antivirus solutions, and how they collect and analyze telemetry from endpoints to detect and respond to threats. The room culminates in a hands-on investigation of a realistic alert using a simulated EDR interface, reinforcing how SOC analysts use EDR data in real-world triage.

---

## Key Concepts Covered

- **EDR basics** — what an EDR is, how it works, and why it is essential in modern endpoint security
- **EDR vs Traditional Antivirus** — understanding the limitations of legacy AV and how EDR extends detection and response capabilities beyond signature-based methods
- **EDR architecture** — examining the components that make up an EDR solution and how they interact
- **Telemetry collection** — analyzing the types of data EDRs collect from endpoints including process activity, file system changes, network connections, registry modifications, and user behavior
- **Detection and response capabilities** — understanding how EDRs detect threats in real time and the response actions available to analysts
- **Realistic alert investigation** — applying EDR knowledge to investigate a simulated endpoint alert end-to-end
