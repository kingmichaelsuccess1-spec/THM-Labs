## Description
This room explores how systems serve as attack vectors in the modern digital world. While SOC analysts don't typically manage systems directly, understanding system-level attacks and communicating them to the IT department is essential for building a strong security posture.

## Key Concepts Covered
- The role of systems in a modern digital environment
- Unpatched software — exploitable vulnerabilities from outdated systems
- Misconfigurations — setup errors that expose systems to attack
- Malware — types, delivery methods, and system-level impact
- Supply chain attacks — compromising trusted third-party software and vendors
- Two realistic hands-on attack scenarios

## SOC Analyst Takeaways
- Threat Awareness: Staying current on system-level threats
- Cross-Team Collaboration: Sharing threat intelligence with IT teams
- Detection: Indicators tied to unpatched systems, misconfigurations, malware, and supply chain compromise

## Skills Demonstrated
- System-level attack surface and threat category knowledge
- Communicating system vulnerabilities in a SOC context
- Practical scenario-based application of concepts
- Cross-functional awareness supporting IT and security collaboration
