## Description
This room introduces the reference materials and structured tools that SOC analysts rely on during active investigations. It covers SOC investigation workbooks, asset inventory lookups, and corporate network diagrams — all critical context for making informed triage decisions. The room also includes hands-on practice building investigation workflows inside a simulated interactive interface.

## Key Concepts Covered
- SOC investigation workbooks — structured investigation steps to stay consistent
- Asset inventory — where to find it and how to use it for accurate triage
- Corporate network diagrams — understanding layout to contextualize alerts
- Workflow building — constructing investigation workflows in a simulated SOC interface

## SOC Analyst Takeaways
- Workbooks: Using structured guides to stay thorough during alert review
- Asset Inventory: Referencing asset data to determine scope and criticality of an alert
- Network Diagrams: Reading corporate network layouts to place alerts in context
- Workflow Building: Constructing repeatable SOC workflows for efficient investigations

## Skills Demonstrated
- Navigating and applying SOC investigation workbooks
- Using asset inventory lookups to support triage decisions
- Interpreting corporate network diagrams during investigations
- Building structured investigation workflows in a simulated SOC environment
